* B站课程选择的原则是**优质>新鲜>有影响力**：
    * “优质”是我根据视频制作质量、视频内容（目录）等因素主观判断得出的结论；
    * “新鲜”是指视频系列的活跃时间（是否为近期发布、是否还在更新）；
    * “有影响力”是指视频发布者的粉丝数、系列播放数等；

* 本教程的制作方式如下：
    * 由我先过一次系列视频中的各个分p视频，并形成记录pdf文档，存储在该分p在本地（本教程）对应的文件夹中；
    * 对于值得动手操作 / 有补充讲解的分p视频，我会专门录屏形成MP4文件，同样存储在该分p在本地（本教程）对应的文件夹中；
