## 生成新变量
* 取两列向量，进行逐元素相乘，并把结果赋给数据框（取了一个新名字，等于在原数据框中新建了一列）：
    ```R
    tianmao['total_sales'] <- tianmao$current_price * tianmao$month_sales_count
    # 当然，也可以这么创建：
    tianmao$total_sales <- tianmao$current_price * tianmao$month_sales_count
    ```
* 又用了`ifelse()`进行了面向向量的批量化操作。这里用了一个`ifelse()`函数的嵌套，实现了双重判断：
    ```R
    tianmao['price_tag'] <- ifelse(tianmao$current_price < 1000, 'low price',
                                ifelse(tianmao$current_price <= 2000, 'fair price',
                                'high price'))
    ```

## 变量重命名
* 使用`names()`函数可以查看数据框的列名。
* 此外，可以直接对`names()`的返回结果进行赋值，实现列名的修改，例如
    ```R
    names(tianmao)[1] <- 'new_name'
    ```
* 介绍了一个新的语法，`%in%`，作用判断一个元素是否存在于另一个集合里，返回 `TRUE`/`FALSE`，例如
    ```R
    元素 %in% 集合
    # 例子1
    3 %in% 1:5   # TRUE
    # 例子2 - 向量批量判断
    c(1,6,3) %in% c(1,2,3)  # TRUE FALSE TRUE
    ```
    * 注意！这个判断运算符非常灵活！`%in%`前后的元素可长可短！就课上的例子而言，
    ```R
    names(tianmao) %in% "discount"
    > [1] FALSE FALSE FALSE FALSE FALSE FALSE FALSE  TRUE FALSE
    "discount" %in% names(tianmao)
    > [1] TRUE
    ```
* 讲了一个比较复杂的重命名逻辑，通过`names(tianmao) %in% "discount"`语句，以逻辑向量的形式，获取想要重命名的那一列（`discount`）的位置，并用这个逻辑向量进行索引，将索引到的列名修改成新名称：
    ```R
    names(tianmao)[names(tianmao) %in% "discount"] = 'zhekou'
    ```

## 提取数据框子集
* 可以用名称、整数来索引数据框以获取数据框子集，也可以用负索引来剔除部分列，以获取数据框子集。
* 提到了一个小知识点，`!`是布尔值取反操作
* 讲了一个比较复杂的删除部分列的逻辑，通过`ol1_idx <- names(tianmao) %in% col1`以逻辑向量的形式获取想删除的列的索引，取反后提取子集。
    ```R
    col1 <- c('new_name', 'shop_name', 'original_price')
    col1_idx <- names(tianmao) %in% col1
    new_data2 <- tianmao[, !col1_idx]
    ```
* 提到了一个提取子集的函数，`subset()`。这个函数是常用的数据框 / 向量子集提取函数，作用**按条件筛选行**、**按列名提取列**，基本语法如下：
    ```R
    subset(x, subset, select, drop = FALSE, ...)
    ```
    参数的含义如下：
    * `x`：	待操作的对象（通常是数据框 data.frame，也支持向量、矩阵）；
    * `subset`：**行筛选条件**，逻辑表达式，满足条件的行被保留；
    * `select`：**列筛选条件**，指定要保留 / 删除的列，支持列名、向量、`-` 符号；
    * `drop`：仅矩阵 / 数组使用，是否降维，默认 `FALSE`。
