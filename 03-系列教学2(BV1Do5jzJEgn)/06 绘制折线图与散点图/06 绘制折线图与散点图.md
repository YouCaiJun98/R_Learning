* 介绍若干低级绘图函数，`points()`、`lines()`、`abline()`。此外，介绍将多条曲线/散点绘制到同一张图像上的方法。

## 低级绘图函数
* `abline()`函数是给现有图形添加直线的函数，专门用来在散点图、折线图上加参考线、回归线、竖线、横线。核心用法如下所示：
    ```R
    # 1. 加斜线：y = a + bx
    abline(a = 截距, b = 斜率)

    # 2. 加水平线（y = 固定值）
    abline(h = 数值)

    # 3. 加竖直线（x = 固定值）
    abline(v = 数值)

    # 4. 加回归直线
    abline(回归模型)
    ```
* 介绍了一个新的类型转换函数，`as.POSIXct()`，它的作用是将字符串转换成时间，例如：
    ```R
    # 文本时间（不能直接计算）
    time_str <- "2025-01-01 12:00:00"

    # 转换成标准时间格式
    time_real <- as.POSIXct(time_str)

    # 查看类型
    class(time_real)
    ```
* 介绍了`lines()`。它是在已有的图上，追加画一条折线的函数。它不能单独画图，必须跟在`plot()`后面使用。使用示例如下：
    ```R
    # 先画底图
    plot(x, y1, type = "l")

    # 再叠加第二条线
    lines(x, y2, col = "red")
    ```
* 介绍了一个获取数值范围的函数，`range()`，返回一组数据中的最小值和最大值，组成一个长度为 2 的向量。
    ```R
    x <- c(3, 8, 1, 12, 5)
    range(x)
    # 输出：1 12
    ```
* 介绍了`points()`函数，这个函数的作用是在已经画好的图上，追加绘制点（散点）。它也不能单独画图，必须跟在 plot() 后面用。常用的语法如下所示：
    ```R
    points(x, y, pch = 点形状, col = 颜色, cex = 大小)
    ```
    使用示例如下：
    ```R
    plot(stock1$x, stock1$y, col = "blue")
    #  叠加另一类点
    points(stock2$x, stock2$y, col = "red")
    ```
* 介绍了一个逐点控制颜色的绘图方式，它的逻辑是，用`stock$label`作为索引，生成一个和总点数等长的颜色向量，并用这个向量给图像中的点添加颜色。
    ```R
    plot(stock$SH_closing_price, stock$investor_confidence_index,
        col = c('blue', 'green')[stock$label])
    ```
* 介绍了`matplot()`函数，这个函数是专门用来给矩阵 / 多列数据批量画多条线 / 多个点的函数，一行代码画多条曲线。它的基本用法如下：
    ```R
    matplot(x, y矩阵, type = "l")
    ```
    其中，
    * `x`：横坐标
    * `y`：多列数据矩阵（每一列画一条线）
    * `type = "l"`：画折线（默认是点）
* 补充：UP主提到了“泛型函数”，什么是“泛型函数”？
    * 同一个函数名，根据传入对象的类型不同，自动执行不同的代码。
    * 例如，函数名字一样，都是`plot()`，你传给它 lm 模型，它就画诊断图；你传给它数据框，它就画散点图矩阵；你传给它向量，它就画折线 / 散点。
