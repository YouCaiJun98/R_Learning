## 向量的特点
* 向量是`R`语言最基础的数据类型。定义是相同类型元素组成的有序集合。
* `R`中不存在标量（单个数值）的概念，标量也是向量（长度为1）。

向量有以下的特点：
1. 有序
    * 向量的元素有对应的位置（下标）；可以用下标访问到对应位置的元素。
        ![](https://raw.githubusercontent.com/YouCaiJun98/MyPicBed/main/imgs/20260410122308048.png)
        * 比如可以用下标"2"来访问到"8.2"这个元素。
2. 元素类型统一
    * 常用的有数值（numeric），字符（char），逻辑值（bool）。
    * 一般情况下，如果向量中有不同类型的元素，`R`会将它们自动转换成相同类型的。

## 创建向量
* 在创建字符型向量时，可以用单引号也可以用双引号。
* 创建一个混合型的向量时，`R`会将它们统一成一样的类型：
    ```R
    c(1, "a", TRUE)
    > "1", "a", "TRUE"
    ```
    * 这里遵循的转换规则是，「数值→字符」，「逻辑→数值 / 字符」。
* 介绍了因子向量（`factor()`），它在`R`中的起到了分类变量的作用。

## 向量的索引
* 在向量后用方括号做索引，目的是得到向量的子集。
* 索引向量可以是整数向量、负整数向量、逻辑向量、字符串向量。
    ```R
    num1[1]           # 整数索引
    num1[c(1, 3)]     # 向量索引
    num1[c(T, F, T)]  # 逻辑索引
    num1[-2]          # 负数索引
    num3[c('first', 'third')] # 字符索引
    ```
* 介绍了`R`中求余的符号，`%%`
    ```R
    23 %% 2
    > 1
    ```

## 元素的修改
* 在索引的基础上重新赋值。既可以只对一个元素修改，可以同时修改多个元素。

## 查看变量类型
* 可以用`class()`函数查看变量类型。

## 类型的检查与转换
* 检查变量是否是字符型变量，使用`is.character()`函数。
* 将变量转换成字符型的，使用`as.character()`函数。
* 当然，也可以对其他类型的变量进行检查与转换，例如，对于数值型变量，使用`is.numeric()`/`as.numeric()`进行检查/转换；对于逻辑型变量，使用`is.logical()`/`as.logical()`进行检查/转换。
    * 这里提到了一个小规则，在将数值型向量转换成逻辑型时，`0`会被转换成`FALSE`，而其他的数据会被转化成`TRUE`；而将字符类型转换成逻辑型，它们全会被转换成`NA`。

## 特殊数值
* `NA`表示缺失值。
    * 检查缺失值使用`is.na()`函数。这个函数会把`NA`和`NaN`都识别为缺失。
* `NaN`表示无意义的数（比如对负数开根号，但这不是复数吗）。
* `Inf`表示正无穷。
    * 检查是否是有限大的数，用`is.finite()`函数。
    * 检查是否是无限大的数，用`is.infinite()`函数。
* `-Inf`表示负无穷。
* `NULL`表示不存在。
