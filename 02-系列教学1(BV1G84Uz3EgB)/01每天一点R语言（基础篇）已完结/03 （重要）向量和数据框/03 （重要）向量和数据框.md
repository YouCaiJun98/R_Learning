## 什么是向量
* 是一个`数组`（什么是`数组`？），如下框所示的公式，就是`combine`了五个数字，形成了一个`向量`：
    ```R
    c(1, 2, 3, 4, 5)
    ```
* 这样做以后，这堆数字就会以一个集合的形式出现在`R`的临时内存中；
* `R`中对数据的操作都是以向量为单位的（向量化操作）；
* 下面的例子，就是对这5个数同时+1：
    ```R
    c(1, 2, 3, 4, 5) + 1
    ```
* 在`R`语言中，在一行代码之前加"#"，解释器就会把这行代码给忽略掉（什么是解释器？什么是注释？）。
* 在实际操作中，可以按`Ctrl + Shift + C`来注释掉选中的代码；对于已被注释的代码，可以再按`Ctrl + Shift + C`来取消注释。

## 如何扩展向量
* （视频6:53）扩展向量，就是在原向量里再加几个元素。
    * 方法一：重新赋值
        ```R
        A <- c("A", "B", "C", "D", "E", "F")
        ```
    * 方法二：用原来的元素代表自己：
        ```R
        A <- c(A, "E", "F")
        ```

## 如何提取数据框的列
* （视频11:12）这里提到了一个概念，在已经生成数据框后，修改用来初始化数据框的变量，**数据框本身不会变化**：
    ```R
    Df <- data.frame(A, B, C)
    B <- B + 1
    Df == Df! # Df中的B不会随着之前的B的变化而变化！
    ```
* 用下面的符号来提取数据框中的元素：
    ```R
    Df$B
    ```
* 这样，就可以修改数据框中的元素了：
    ```R
    Df$B <- Df$B + 1
    ```

## 如何扩展数据框的列
* （视频12:21） 和之前扩展向量的方法很像：
    ```R
    D = c(6, 5, 4, 3, 2, 1)
    Df <- data.frame(Df, D)
    ```
* 或者可以这么做，直接在Df里追加一个向量：
    ```R
    Df$E <- c(6, 5, 4, 3, 2, 1)
    ```

## 数据框的行名和列名
![](https://raw.githubusercontent.com/YouCaiJun98/MyPicBed/main/imgs/20260323125009991.png)
* （视频14:31）可以用下面的方式提取数据框的“行名”：
    ```R
    Df[3] # 这样是有问题的，这是提取了第3个元素，其实是第3列！（也叫数据框的“一维索引”）
    ```
* 这样可以提取数据框中的某个元素：
    ```R
    Df[2, 4] # 这样是提取了数据框的第二行第四列的元素
    ```
* 如果上述的位置中空了一个，就是取第x行 / 第x列：
    ```R
    Df[, 4] # 这是取第4列
    Df[2, ] # 这是取第2行
    ```
* （视频18:21）UP主说的细微的差别是什么？（应该是按照行向量/列向量来组织的区别）。
* 当然，也可以直接这么取：
    ```R
    Df$C
    ```
* 可以用这样的方式给行起个名字：
    ```R
    rownames(Df) <- c('a', 'b', 'c', 'd', 'e', 'f')
    ```
* 然后，就可以按这样的方法取某个名字对应的那一行了：
    ```R
    Df['c', ]
    ```
* 用起名来取数据的好处是，你可以不用知道具体是哪一行，只取那个名字对应的行就可以了。
* 当然，也可以这么取某一列的数据：
    ```R
    Df[, 'C']
    ```
