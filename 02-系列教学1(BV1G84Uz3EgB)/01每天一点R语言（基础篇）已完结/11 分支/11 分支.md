## if else语法
* 基本语法如下：
    ```R
    if (条件语句) {
        执行语句
    }
    ```
* 示例：
    ```R
    # 如果2 > 1，则令a的值为1
    if (2 > 1) {
        a <- 1
    }
    ```
* `if`可以和`else`搭配，表示如果`条件语句`成立，则`执行语句1`的内容，否则，则`执行语句2`。语法如下：
    ```R
    if (条件语句) {
        执行语句1
    } else {
        执行语句2
    }
    ```



## ifelse语法
* `if`和`else`还可以写到一块，这样它就是一个整体（可以视为一个函数），语法如下：
    ```R
    ifelse(条件语句, 执行语句1, 执行语句2)
    ```
    * 如果条件语句成立（`T`），则执行语句1，否则，则执行语句2
* 当然，还可以这么写，因为`ifelse`是有**返回值**的（什么是返回值？）
    ```R
    a <- ifelse(2 < 1, 2, 3)
    ```

## 条件判断控制下的循环
* UP主又提了一个新的操作符，大于等于，对应`>=`；同理，小于等于是`<=`。
* UP主又又提了一个新的操作符，`&`:
    * `&`是向量之间的与运算；
    * `&&`是标量之间的与运算。
    * 同理，`|`是向量之间的运算，`||`是标量之间的运算。
* 但实际上，下面的操作也是正确的：
    ```R
    for (i in 1:100) {
        if (i < 18 && i > 5) {
            a <- c(a, i)
        }
    }
    ```
    * 这是因为，`i < 18`和`i > 5`分别返回了一个`bool`值，这两个`bool`值都是标量，所以可以用`&&`！
* 最后，讲了一个更简单的子集提取方法（没有必要用循环！用两个条件判断返回`bool`向量，做索引即可）：
    ```R
    b <- 1:100 
    b[b<=70 & b >=18]
    ```
