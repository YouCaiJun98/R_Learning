## 生成列联表
* 检查案例中基线是否是齐的（是否有差异），可以用`table()`生成列联表，如示例中的：
    ```R
    table(df$Group, df$Path)
    ```

## 卡方检验
* 无序的分类变量适合用卡方检验；
* 将列联表输入卡方检验`chisq.test()`，即可进行卡方检验：
    ```R
    chisq.test(table(df$Group, df$Path))
    ```

## Fisher检验
* 当样本量比较少的时候，选用Fisher精确检验，而不要用卡方检验：
    ```R
    fisher.test(table(df$Group, df$Path))
    ```
* UP主进行了一次类比，就好比秩和检验之于t检验，能用卡方检验的地方一定能用Fisher检验（可以这么理解，能用条件约束更大的检验，就一定能用约束更宽松的检验）。

## 补充：卡方检验与Fisher检验
### 卡方检验
* 卡方检验（Chi-square test, $\chi^2$检验）是统计学里最常用的**非参数检验**（不要求正态分布）之一，专门用来分析分类数据（计数数据），可以将它理解为，用来判断**实际观察到的频数**和**理论期望的频数**是否一致的检验。
* 核心用途包括：
    * 拟合优度检验（单样本）：看一组分类数据的分布是否符合某个理论分布（如正态、均匀、遗传比例等）；
    * 独立性检验（最常用）：看两个分类变量之间**是否相互独立**，例如：吸烟与肺癌是否相关；性别与专业选择是否独立；
    * 齐性检验：看多个总体的某分类变量分布是否相同。
* 适用的数据类型包括：
    * **计数数据 / 频数表**；
    * 分类变量：名义变量（男 / 女、是 / 否）、有序分类；
* 基本流程：
    1. 计算观察频数$O$
    2. 计算期望频数$E$
    3. 构造卡方统计量$\chi^2=\sum\frac{(O - E)^2}{E}$
    4. 比较$\chi^2$与临界值，得到$p$值，若$p<0.05$，则拒绝原假设，说明差异显著/不独立。

### Fisher精确检验
* Fisher精确检验（Fisher's Exact Test）**只适用于2×2列联表**（两组×两类），当卡方检验不能用时，就用Fisher精确检验。
* 当2×2列联表满足下面任一情况时，卡方检验是近似检验，这种情况下不准，Fisher是精确计算概率，更可靠：
    * 样本量小
    * 期望频数 < 5 的格子太多
    * 总样本量 $N < 40$
* Fisher精确检验的特点包括：
    * 只适用于**2×2列联表**（两组 × 两类）
    * 不需要大样本近似
    * 结果是精确p值，属于非参数检验。
* Q：为什么视频的例子中，列联表明明是2x4的，却还是可以用fisher检验？
    * A：传统意义上的Fisher精确检验确实是为2×2列联表设计的。但是，`R`语言对`fisher.test()`函数做了扩展，支持r × c列联表（r行c列，r,c ≥ 2）！
    * r × c扩展版Fisher检验将超几何分布推广到多元超几何分布（我也不懂😵），逻辑和经典2×2 Fisher检验完全一致，固定行和、列和不变，枚举所有可能的列联表，计算每个表的精确概率，累加得到p值。
    * 结论：本质上，`R`实现的是**广义Fisher精确检验**，不再局限于2×2表！
