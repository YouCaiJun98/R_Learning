## for循环
* for循环的基本语法如下：
    ```R
    for (i in vec) {
        <loop_struct>
    }
    ```
    * 上面的`i`是一个元素，是对向量`vec`中的元素做**遍历**（即对于每个元素，都做循环体的操作）
    * 上面的`vec`是一个向量，其中的元素类型不用太关心，可以是数值，可以是字符，等等等等。
* UP主又偷摸着提了一个之前没提到的知识点，即通过“切片”来取一个向量中的部分元素：
    ```R
    a[1:10] # 取a中1-10号元素
    ```

## while循环
* 适用于未知循环次数，满足条件就循环，条件不满足则停止。
    * 注意：必须在循环内更新条件变量，否则会死循环。
    ```R
    # 语法
    while (条件) {
        执行代码
        更新条件
    }

    # 示例：i小于5时循环
    i <- 1
    while (i < 5) {
        cat("当前i：", i, "\n")
        i <- i + 1  # 必须更新，否则死循环
    }
    ```

## repeat循环
* 无条件循环，必须配合`break`停止，否则无限执行。
    ```R
    # 语法
    repeat {
        执行代码
    if (终止条件) break
    }

    # 示例：累加至大于10停止
    sum <- 0
    repeat {
        sum <- sum + 2
        cat("当前和：", sum, "\n")
    if (sum > 10) break  # 终止循环
    }
    ```
