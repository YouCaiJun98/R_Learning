## 简单运算符
* 单个数之间的运算符如下：
    ```R
    + # 加
    - # 减
    * # 乘
    / # 除
    ^ # 乘方
    ```
* 向量之间也可以运算：
    ```R
    c(1, 2, 3, 4) + 1 # 向量的每个元素都加1
    c(1, 2, 3, 4) + c(4, 3, 2, 1) # 两个向量之间做加法
    ```
* 如果两个向量不等长，短的那个就会被复制若干份，对齐到长的那个：
    ```R
    c(1, 2, 3, 4) + c(4, 3) # 等效于c(1, 2, 3, 4) + c(4, 3, 4, 3)，结果是c(5, 5, 7, 7)
    ```
    * 但是，**不推荐进行不等长向量之间的运算！** 最好还是一个向量加一个标量，或者两个等长的向量进行运算！


## 数学函数
* 有几个常用的数学函数：
    * 绝对值
        ```R
        abs()
        ```
    * 平方根
        ```R
        sqrt()
        ```
    * 对数
        ```R
        log()
        ```
    * 指数运算
        ```R
        exp() # e^n
        ```
