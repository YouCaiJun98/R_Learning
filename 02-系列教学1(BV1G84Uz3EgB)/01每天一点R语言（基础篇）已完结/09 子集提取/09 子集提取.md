## 子集提取方法1
* UP主讲了一个快速创建向量的方法，用`:`：
    ```R
    d <- 1:5 # 等效于c(1, 2, 3, 4, 5)
    ```
* 提取方法1就是按照索引提取，和之前的课程讲的一样：
    ```R
    df[1, 3] # 第一行第三个元素
    df[1, ]  # 第一行元素
    df[, 1]  # 第一列元素
    ```
* 还有就是按照列名、行名提取：
    ```R
    df$a      # 取a列
    df[, "a"] # 取a列
    ```

## 子集提取方法2
* 优势：数据量大的时候方式2占优。
* 用bool向量提取元素（这一点和上节课最后提到的一样，记得吗？）
    ```R
    b <-    c('A', 'B', 'C', 'D', 'E')
    #          ↑    ↑    ↑    ↑    ↑
    idx_b = c( T,   T,   F,   F,   T)
    b_ <- b[idx_b] # c('A', 'B', 'E')
    ```
* 对于数据框，也可以用bool向量提取元素：
    ```R
    idx_row <- c(T, T, F, F, T)
    idx_col <- c(F, T, T)
    df[idx_row, idx_col]
    ```

## 一个应用案例
* 先条件判断获得一个bool变量，再提取数据框中对应的数据：
    ```R
    df[df$d>2.5, ] # 按照df$d进行判断，取该值大于2.5的行
    df[df$b!='B', ] # 按照df$b进行判断，取字符不为"B"的行
    ```
