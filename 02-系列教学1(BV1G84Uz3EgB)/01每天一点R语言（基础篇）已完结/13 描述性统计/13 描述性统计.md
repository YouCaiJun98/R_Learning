## 数值型
* 包括均值、标准差、中位数、四分位距
    * 均值：
        ```R
        mean()
        ```
    * 标准差：
        ```R
        sd()
        ```
    * 中位数：
        ```
        median()
        ```
    * 四分位距：
        ```R
        quantile()
        ```
    * 注意：如果被计算的向量中存在`NA`，那么这个`NA`就会传播，导致最后的结果乱掉；
        * 此时，可以在这些内置函数中，将`na.rm`设置为`T`，这样在计算时就会自动去除`NA`值了。

## 分类变量
* 包括个数、百分比
    * 元素个数统计：
        ```R
        table()
        ```
        * 如果想把`NA`也统计进来，可以设置`useNA = "ifany"`
        * `table()`返回的结果是个特殊的**数据结构**（什么是数据结构？），一般来说要先从中提取特定的元素再进行运算。
        * **数据结构**的例子：
            ```C
            struct Student {
                string name;
                int age;
                float score;
            }
            ```
    * 如果想要查看某个内置函数的说明，可以用下面的命令：
        ```R
        help(<func_name>)
        ```
        * 当然，还有更简单的方法！可以直接`?<func_name>`！
    * 统计向量中元素的总数：
        ```R
        length()
        ```
