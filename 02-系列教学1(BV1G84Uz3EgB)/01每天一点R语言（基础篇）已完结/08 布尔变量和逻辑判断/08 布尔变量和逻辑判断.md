## 什么是布尔变量
* 对（True）或者错（False）
* 在`R`中，`T`和`F`分别表示了两个布尔值：
    ```R
    T # TRUE
    F # FALSE
    ```
* 虽然下面的赋值是合法的，但是尽量不要这么做（不然你就没办法表示布尔值了）：
    ```R
    T = c(1, 2, 3)
    ```
* 


## 如何得到布尔变量（R语言的逻辑运算）
* 数学运算，通过不等式/等式来得到一个布尔值（判断给定的式子是否成立）：
    ```R
    2 > 1 # T
    2 < 1 # F
    2 == 1 # F
    2 != 1 # T 当心中文字符！
    ```
* 逻辑运算，通过“与或非”来判断事件是否成立，进而生成一个布尔值
    ```R
    A && B # 与- A与B同时为真？如果A、B均为真，则这个表达式返回T，有一个为假就是F
    A || B # 或 - A和B中只要有一个为真，那整个表达式的返回值就是T，如果都是假，那就是F
    !A     # 非 - 非真就是假，非假就是真
    ```

## 一个例子
* up主提到了一个会生成布尔值的例子，用到了`is.na()`这个函数：
    ```R
    a = c(12, 2, 3, NA, 5)
    ```
    * `is.na()`，传入一个向量，返回一个布尔值类型的向量（其中的元素都是`T`或者`F`，如果对应元素是`NA`，那就返回`T`，否则就是`F`）
* 还提到了一种用布尔变量索引的方式（**和之前的方式不一样！**），即：
    ```R
    a[!is.na(a)]
    ```
    * 这里，`is.na(a)`返回了一个和`a`等长的布尔向量（这里的情况是`c(T, T, T, F, T)`），接着，就按照这个向量“按图索骥”，取`a`中对应位置为`T`的元素，生成一个新的向量：
    ```R
    !is.na(a)      # c(T,  T, T, F,  T)
    #                  ↓   ↓  ↓  ↓   ↓
    a             # c(12,  2, 3, NA, 5)
    a[!is.na(a)]  # c(12,  2, 3,     5)
    ```
