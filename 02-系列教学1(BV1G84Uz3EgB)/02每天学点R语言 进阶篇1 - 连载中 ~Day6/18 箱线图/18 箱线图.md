## 箱线图绘制
* UP主说，只要是个连续型变量，都可以将它作为y轴绘制箱线图。绘制箱线图时，不要求数据服从正态分布。
* 和散点图不一样的是，这里要传入一个`fill`字段，而不是`color`（但它们都是用来控制颜色的，`fill`的意思是把box填充成什么颜色）。
* 课程中的示例代码如下：
    ```R
    ggplot(medical, aes(x = disease, y = sbp, fill = disease)) + 
        geom_boxplot()
    ```

## 补充：如何加标题
* 既然UP主提了一句加标题，那我们就试一下：
    ```R
    ggplot(medical, aes(x = disease, y = sbp, fill = disease)) + 
        geom_boxplot() + 
        labs(caption = "底部备注：仅用于教学示例") +
        theme(
            plot.caption = element_text(
            size = 10,        # 字号
            color = "gray50",  # 颜色
            hjust = 0,        # 0左对齐 / 0.5居中 / 1右对齐
            face = "italic"   # 斜体
            )
        )
    ```
* 画图结果如下：
