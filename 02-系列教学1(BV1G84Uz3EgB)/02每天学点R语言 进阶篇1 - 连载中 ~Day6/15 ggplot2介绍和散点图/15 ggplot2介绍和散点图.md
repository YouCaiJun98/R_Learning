## ggplot2介绍
* `ggplot2`是`R`语言最主流、逻辑统一的绘图包，可以画散点图、折线图、柱状图、热力图、分组图等，是数据分析可视化标配。
* `ggplot2`的核心思想是图层叠加（语法图形），一张图 = 底数据 + 坐标轴 + 几何图形 + 美化图层，层层往上叠，改图不用重写全部代码。
* 固定的绘图骨架是：
    ```R
    ggplot(数据, aes(x=X轴列, y=Y轴列, color=分组列)) +
        geom_xxx()   # 画什么图形
    ```

## ggplot2散点图
* UP主提到了一个函数`set.seed()`，它的作用是设置随机数种子。在保证随机数种子一定的前提下，可以复现实验结果（否则，每次的随机操作，比如采样等都会有所不同）。
* 在生成随机数的时候，用到了`runif()`这个函数，它是`uniform`的缩写，即按照均匀分布采样数据。具体的语法如下：
    ```R
    runif(n, min, max) # n是生成随机数的个数，min是随机数下限（默认为0），max是随机数上限（默认为1）
    runif(3, 2, 5) # 生成3个2~5的随机数
    ```
* `rnorm()`之前介绍过，是生成服从正态分布的数据点。
* `round()`之前也介绍过，是对小数做舍入。
* `sample()`函数是，从一组数据里**随机抽样**（抽几个、抽完放不放回去都能控制），基本的语法是：
    ```R
    sample(x, size, replace = FALSE) # x是要抽样的数据源（向量/数字），size是抽多少个，replace是是否有放回抽样（默认 FALSE，不放回）
    # 举个例子：从 1:10 里随机抽 3 个
    sample(1:10, 3)
    ```
* 课程中的绘图代码如下：
    ```R
    # 横轴是年龄，纵轴是收缩压，需要什么绘制的内容就不断往上“+”
    medical %>% ggplot(aes(x = age, y = sbp)) +
    geom_point()
    ```
