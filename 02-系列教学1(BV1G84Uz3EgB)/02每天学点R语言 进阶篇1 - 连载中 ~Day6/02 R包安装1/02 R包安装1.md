## 搜索R包
* 在搜索引擎中找到想要安装的`R`包：
    ![](https://raw.githubusercontent.com/YouCaiJun98/MyPicBed/main/imgs/20260329132833114.png)
    * "CRAN"代表这个包可以通过install package的方法安装（即下载一个安装包，在本地安装）

## 安装R包
* 使用以下命令来安装`R`包：
    ```R
    install.packages("<包名>") # 例如，install.packages("survival")
    ```

## 加载包内功能
* 方法1，使用`<包名>::函数`的方式来调用包内函数，例如：
    ```R
    survival::coxph()
    ```
    * 一般不会这么用，除非遇到**命名冲突**的问题（Q：什么是命名冲突？A：不同包里有同名函数，需要做区分。）
* 方法2，使用`library()`函数来加载这个包，这样可以把这个包的函数一口气全加载进来，例如：
    ```R
    library(survival)
    ```
