## 颜色设置方式
* 方式1 - 在坐标轴里设置：
    ```R
    medical %>% ggplot(aes(x = age, y = sbp, color=gender)) + # color在aes里
        geom_point()
    ```
    * 特点是做出来的图像都会共享这种颜色设置（比如画不止一张图，都会按照这种颜色模式）
* 方式2 - 在具体的绘图函数里设置颜色：
    ```R
    medical %>% ggplot(aes(x = age, y = sbp)) +
        geom_point(aes(color=gender))
    ```
    * 特点是可以按图幅控制，每张图都可以有不一样的颜色设置。
    * 当然，也可以不用`aes()`包裹，而是直接设置（这样会破坏颜色与变量的对应关系，还是建议在坐标轴里设置）：
    ```R
    medical %>% ggplot(aes(x = age, y = sbp)) +
        geom_point(color="red")
    ```
* 使用`colors()`函数查看色板，也可在这个网站上查询颜色：https://r-charts.com/colors/

## 补充：按变量设置颜色
* 可以先构建一个变量到颜色的映射向量，再根据映射结果画图：
```R
my_color <- c(
  "Male"   = "#E74C3C",   # 红
  "Female" = "#3498DB"    # 蓝
)
medical %>% ggplot(aes(x = age, y = sbp, color=gender)) +
  geom_point() + 
  scale_color_manual(values = my_color)
```
